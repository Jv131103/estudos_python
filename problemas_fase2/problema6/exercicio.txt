EXERCÍCIO 3 — Remover Duplicatas In-Place (Sem usar set)

Dada uma lista não ordenada, remover duplicatas modificando a própria 
lista, sem criar outra lista auxiliar do mesmo tamanho.

Exemplo:

    entrada = [3, 1, 2, 3, 4, 1, 5]
    saida esperada: [3, 1, 2, 4, 5]


Restrições:

    Não usar set()

    Não usar dict

    Não usar list comprehension

    Deve modificar a lista original
